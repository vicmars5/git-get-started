# git get started

Iniciar proyecto de git:

``` Shell
git init
```

Hacer commit
``` Shell
git commit -m [mensaje]
```

Ver ramas
``` Shell
git branch
```

Nueva rama
``` Shell
git branch [nombre-rama]
```

Cambiar de rama
``` Shell
git checkout [nombre-rama]
```

Traer cambios de rama. **Nota**: Debes de estar en la rama a la que quieres traer los cambios.

``` Shell
git merge [nombre-rama]
```
